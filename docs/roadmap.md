# Roadmap

## v1 release candidate

- Stable mock-first support chat and admin shell.
- Free-tier Render, Vercel, and Supabase deployment path.
- RBAC, audit logging, redaction, provider routing, prompt lifecycle, action runtime, connectors, knowledge ingestion, RAG foundation, workflows, and smoke tests.

## v1.1 candidates

- Real provider transport plugins.
- Stronger identity integration.
- More import/export tools for knowledge.
- Improved dashboard filtering and demo screenshots.

## Later, explicitly out of current scope

- Billing.
- SSO.
- Kubernetes.
- Redis/queue infrastructure.
- Heavy vector database dependency.
